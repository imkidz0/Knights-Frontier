# [KF-41] AV/EDR 다계층 탐지 메커니즘 종합 분석 made by Theo
## 미션 소개
- **미션 목표:** 단일 악성 행위에 대해 정적/동적/휴리스틱 등 다계층 탐지가 어떻게 작동하는지 종합 분석, 제품별 탐지 역량 비교
- **수행 과정:**
1. 분석 대상 행위 선정 (예: 프로세스 인젝션, 파일 드롭, 자격증명 접근 등 MITRE 기준 1~2개)
2. 해당 행위 구현체의 탐지 레이어별 분석  
 정적: 시그니처, IAT, 문자열, 엔트로피 등  
 동적: API 호출 패턴, 행위 시퀀스, 메모리 속성  
 휴리스틱: 평판, 출처, 디지털 서명, 파일 경로
3. 각 레이어별 탐지 트리거 조건 실험 및 기록
4. 레이어 간 상관관계 분석 (예: 정적 우회 시 동적에서 걸리는지)
5. 멀티 AV/EDR 비교 테스트
6. 최소 3개 이상 제품 대상 동일 테스트 수행
제품별 탐지 시점, 탐지 레이어, 알림 방식 차이 기록, 제품 간 탐지 공백 영역 식별
- **산출물:** 다계층 탐지 분석 보고서 + 제품별 탐지 비교 매트릭스 + 블로그 포스트
- **최소 인원:** 1명
- **예상 기간:** 2주
- **관련 링크:** https://attack.mitre.org/matrices/enterprise/
